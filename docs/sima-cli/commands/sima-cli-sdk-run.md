# `sima-cli sdk run`

Run a .sima hybrid script with local + container commands.

Parent command: [`sima-cli sdk`](./sima-cli-sdk.md)

## Usage

```bash
sima-cli sdk run [OPTIONS] SCRIPT_PATH
```

## Options

None.

## Arguments

| Name | Description |
| --- | --- |
| `SCRIPT_PATH` | (required) |
