# `sima-cli device`

Discover and manage device(s) for MPK deployment and app lifecycle management purposes, compatible with both PCIe and Ethernet deployment models. Host side only.

Parent command: [`sima-cli`](./sima-cli.md)

## Usage

```bash
sima-cli device [OPTIONS] COMMAND [ARGS]...
```

## Options

None.

## Arguments

None.

## Subcommands

- [`sima-cli device discover`](./sima-cli-device-discover.md): Discover nearby SiMa.ai DevKits via ARP or multicast.
