# `sima-cli playbooks apply`

Apply an installed rule playbook to the current git repository.

Parent command: [`sima-cli playbooks`](./sima-cli-playbooks.md)

## Usage

```bash
sima-cli playbooks apply [OPTIONS] [KIT_ID]
```

## Options

| Name | Description |
| --- | --- |
| `-y, --yes` | Skip confirmation prompt. |

## Arguments

| Name | Description |
| --- | --- |
| `KIT_ID` |  |
