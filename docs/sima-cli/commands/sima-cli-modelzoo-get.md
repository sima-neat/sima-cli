# `sima-cli modelzoo get`

Download a specific model.

Parent command: [`sima-cli modelzoo`](./sima-cli-modelzoo.md)

## Usage

```bash
sima-cli modelzoo get [OPTIONS] MODEL_NAME
```

## Options

None.

## Arguments

| Name | Description |
| --- | --- |
| `MODEL_NAME` | (required) |
