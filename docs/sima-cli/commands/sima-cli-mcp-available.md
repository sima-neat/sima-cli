# `sima-cli mcp available`

List the coding-agent backends sima-cli can register the MCP server with.

Parent command: [`sima-cli mcp`](./sima-cli-mcp.md)

## Usage

```bash
sima-cli mcp available [OPTIONS]
```

## Options

None.

## Arguments
