# `sima-cli sdcard`

Prepare the SD Card as a data storage device for MLSoc DevKit or Modalix Early Access Unit

Parent command: [`sima-cli`](./sima-cli.md)

## Usage

```bash
sima-cli sdcard [OPTIONS] {format}
```

## Options

None.

## Arguments

| Name | Description |
| --- | --- |
| `OPERATION` | (required) |

## Full Help

