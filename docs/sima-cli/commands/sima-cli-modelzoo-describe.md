# `sima-cli modelzoo describe`

Provide information about a specific model.

Parent command: [`sima-cli modelzoo`](./sima-cli-modelzoo.md)

```bash
sima-cli modelzoo describe [OPTIONS] MODEL_NAME
```

None.
