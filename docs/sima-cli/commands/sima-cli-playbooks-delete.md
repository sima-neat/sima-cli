# `sima-cli playbooks delete`

Delete one installed playbook by id, or all with --all.

Parent command: [`sima-cli playbooks`](./sima-cli-playbooks.md)

## Usage

```bash
sima-cli playbooks delete [OPTIONS] [KIT_ID]
```

## Options

| Name | Description |
| --- | --- |
| `--all` | Remove all installed playbooks. |
| `-y, --yes` | Skip confirmation prompt. |

## Arguments

| Name | Description |
| --- | --- |
| `KIT_ID` |  |
