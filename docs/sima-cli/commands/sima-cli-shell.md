# `sima-cli shell`

Start an interactive sima-cli shell.

Parent command: [`sima-cli`](./sima-cli.md)

## Usage

```bash
sima-cli shell [OPTIONS]
```

## Options

| Name | Description |
| --- | --- |
| `-t, --theme` | Colour theme (dark or light). Defaults to the last used, then dark. Switch live with ':theme'. |

## Arguments

None.

## Full Help

```text
Usage: sima-cli shell [OPTIONS]

Options:

                            used, then dark. Switch live with ':theme'.
  --help                    Show this message and exit.
```
