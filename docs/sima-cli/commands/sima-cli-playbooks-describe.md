# `sima-cli playbooks describe`

Show an installed playbook's manifest and document content.

Parent command: [`sima-cli playbooks`](./sima-cli-playbooks.md)

## Usage

```bash
sima-cli playbooks describe [OPTIONS] KIT_ID
```

## Options

None.

## Arguments

| Name | Description |
| --- | --- |
| `KIT_ID` | (required) |
