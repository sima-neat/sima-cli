# `sima-cli network`

Setup Network IP address on the DevKit

Parent command: [`sima-cli`](./sima-cli.md)

## Usage

```bash
sima-cli network [OPTIONS]
```

## Options

None.

## Arguments

None.

## Full Help

```text
Usage: sima-cli network [OPTIONS]

  Setup Network IP address on the DevKit

  This command only works on the DevKit. It allows user to switch between DHCP
  and Static (Default addresses) IP.

Options:
  --help  Show this message and exit.
```
