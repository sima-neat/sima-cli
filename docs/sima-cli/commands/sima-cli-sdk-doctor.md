# `sima-cli sdk doctor`

Run read-only SDK diagnostics.

Parent command: [`sima-cli sdk`](./sima-cli-sdk.md)

## Usage

```bash
sima-cli sdk doctor [OPTIONS] COMMAND [ARGS]...
```

## Options

None.

## Arguments

None.

## Subcommands

- [`sima-cli sdk doctor network`](./sima-cli-sdk-doctor-network.md): Diagnose Ubuntu/Linux host networking for Neat SDK Insight ports.
